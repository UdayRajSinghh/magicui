## Props

| Prop                 | Type              | Default          | Description                                                            |
| -------------------- | ----------------- | ---------------- | ---------------------------------------------------------------------- |
| `children`           | `React.ReactNode` | -                | Content to display over the animated grid                              |
| `gridSize`           | `number`          | `20`             | Size of each grid cell in pixels                                       |
| `gridColor`          | `string`          | `"cyan"`         | Color of the grid and animated cells (cyan, blue, green, purple, pink) |
| `animatedCellsCount` | `number`          | `8`              | Number of animated cells                                               |
| `animationSpeed`     | `number`          | `6`              | Speed of the animation in seconds                                      |
| `glowIntensity`      | `number`          | `0.6`            | Intensity of the glow effect (0-1)                                     |
| `animationDelay`     | `number`          | `3`              | Maximum random delay before animation starts                           |
| `backgroundColor`    | `string`          | `"hsl(0 0% 3%)"` | Background color of the grid                                           |
| `className`          | `string`          | -                | Additional CSS classes                                                 |

## Examples

### Different Colors

```typescript
<AnimatedGridBackground gridColor="blue">
  <YourContent />
</AnimatedGridBackground>
```

### Custom Settings

```typescript
<AnimatedGridBackground
  gridSize={30}
  animatedCellsCount={15}
  animationSpeed={4}
  glowIntensity={0.8}
>
  <YourContent />
</AnimatedGridBackground>
```

### Full Screen Background

```typescript
<AnimatedGridBackground className="min-h-screen">
  <div className="container mx-auto px-4 py-20">
    <h1 className="text-4xl font-bold text-white text-center">
      Welcome to the Future
    </h1>
  </div>
</AnimatedGridBackground>
```
